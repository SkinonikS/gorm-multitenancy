# Security Policy

## Supported Versions

Only the latest major version (`v8`) of gorm-multitenancy is supported. Older versions are not maintained.

## Reporting a Vulnerability

Please use the **"Report a vulnerability"** button in the GitHub Security tab to confidentially disclose any security issues.

## Receiving Updates

Security fixes and announcements will appear on [GitHub Releases](https://github.com/bartventer/gorm-multitenancy/releases) and [Issues](https://github.com/bartventer/gorm-multitenancy/issues).
